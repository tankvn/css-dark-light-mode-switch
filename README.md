### How to create a dark\light mode switch in CSS and Javascript
https://codyhouse.co/blog/post/dark-light-switch-css-javascript
```css
:root {
  --color-bg: #ffffff;
}
[data-theme="dark"] {
  --color-bg: #000000;
}
body {
  background-color: var(--color-bg);
}
```
```html
<body data-theme="dark">
```
